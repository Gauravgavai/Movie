# CineNexus - Movie Exploration Platform 🎬

**CineNexus** is a modern, responsive, and visually appealing web application that allows users to explore movies and content from various film industries, including Bollywood, Hollywood, and Tollywood. This project has been developed with a focus on clean UI/UX and smooth micro-interactions.

---

## 🚀 Features

- **Multi-Industry Dashboards:** Dedicated pages for Bollywood, Hollywood, and Tollywood with customized color themes.
- **Dynamic Search System:** Users can filter movies in real-time based on titles without reloading the page.
- **Interactive Watchlist:** Interactive toggle buttons (`+ Watchlist` / `✓ Added`) to easily add or remove movies from a personal watchlist.
- **Automated Hero Carousel:** A smooth, automated image banner slider on the home page (`index.html`) to showcase featured movies.
- **Responsive Web Design:** Fully optimized layouts tailored for mobile, tablet, and desktop screens.
- **Premium Dark Theme:** Utilizes radial gradients and modern dark UI elements to deliver an immersive cinematic experience.

---

## 🛠️ Tech Stack Used

- **HTML5:** Used for semantic structuring and core webpage layouts.
- **CSS3:** Used for custom gradients, smooth transitions, glassmorphic effects, and responsive styling.
- **JavaScript (ES6):** Powers the real-time search filtering, dynamic UI updates, profile dropdown functionality, and the automated carousel slider logic.
